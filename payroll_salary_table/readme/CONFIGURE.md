On Payroll / Configuration / Salary Table Templates,
defined the different salary tables to be used.

You can provide:

- Name: a meaningful title
- Code: an identifier allowing it to be referenced by salary rules
- Type 1, 2, 3: select one or more Master Data Types to be used to set conditions to
  match.
- Lower Bound and Upper Bound: select if the matching should also compare a given amount
  to a lower and/or upper amout bracket.
